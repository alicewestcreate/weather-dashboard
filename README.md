# weather-dashboard
A weather dashboard that runs in the browser and feature dynamically updated HTML and CSS, using RESTful web APIs. 

## Motivation
- To demonstrate my understanding and ability to use RESTful APIs, using AJAX calls. 
- Generate dynamic page elements based on API responses. 

## What does this problem solve?
The program allows users to search and return the weather forcast for selected cites around the world, and stores historics searches in their local storage.
